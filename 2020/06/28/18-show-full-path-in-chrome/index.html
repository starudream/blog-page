在 Chrome 地址栏中显示完整网址

发表于 2020-06-28 更新于 2020-09-01 分类于 技术 阅读次数： Disqus：

打开 chrome://flags/#omnibox-context-menu-show-full-urls 地址，将 Context menu show full URLs 设置为 Enabled。

重启 Chrome 后，在地址栏右击勾选 总是显示完整网址。
