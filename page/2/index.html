kubernetes LimitRange 解释

发表于 2024-02-19 分类于 技术
  • min: 最小值
  • max: 最大值
  • default 默认 limit 值
  • defaultRequest 默认 request 值

如果 deployment 没有 resources requests 和 limits，那么会使用 defaultRequest 和 default 值，否则使用自定义的值，但是 limits 不能超过 max，requests 不能小于 min。
